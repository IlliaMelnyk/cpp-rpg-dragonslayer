# The Legend of Eternal Dragonslayer 

A 2D command-line Dungeon/RPG game developed in C++. Embark on a journey through mysterious forests and labyrinths to find your destiny in a strange city.

> "Some journeys will never be forgotten!"

## About the Game
You start wandering through a forest, lost and hungry. Suddenly, a giant labyrinth appears. Beyond it lies a mysterious city where your fate changes forever. It is up to you to figure out your new path.

* **Genre:** RPG / Dungeon Crawler
* **Platform:** Windows (Console application)
* **Language:** English
* **Style:** ASCII Art Graphics

## Technologies Used
* **C++** (Core logic)
* **ASCII Generation** (Visuals)
* **IDE:** CLion / Visual Studio
* **Tools:** CMake

## How to Run
1.  Clone this repository or download the source code.
2.  Navigate to the `cmake-build-debug` folder (or build the project using CMake).
3.  Ensure the necessary libraries (`libstdc++-6.dll` and `libgcc_s_seh-1.dll`) are in the same folder as the executable.
4.  Run `TheLegendofEternalDragonslayer.exe`.
